C# Hello World

A simple Hello World application written in C# to demonstrate basic console output.

Prerequisites

  • .NET SDK (version 6.0 or later)
  • Any text editor or IDE (Visual Studio, VS Code, etc.)

Getting Started

Clone this repository to your local machine:

git clone https://github.com/yourusername/csharp-hello-world.git
cd csharp-hello-world

Project Structure

csharp-hello-world/
├── Program.cs
├── HelloWorld.csproj
├── .gitignore
└── README.md

Code Example

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}

Running the Application

To run the application, use the following command in your terminal:

dotnet run
